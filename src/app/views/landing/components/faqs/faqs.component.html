Property Management System or PMS is a system which is used for managing all operations of a Hotel property. From booking to reservation, handling guest request to receiving feedback and managing all other operational works is part of a PMS.
Cloud based PMS is a hotel management software which is hosted on cloud server without investing into any hardware. With this system the PMS can be accessed from anywhere within the property or outside the property through web browser or mobile app. An In property software on the other hand requires a server to be placed inside the property and the PMS is accessible through intranet.
You can select all the modules or select few modules based on the size of your property and requirement. Most of the modules are independent and it can also be integrated with third party software.
